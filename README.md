# ASCII-Art-Generator
Program przetwarza piksel po pikselu obraz i wyświetla wynik w konsoli
> [!NOTE]
> **Aby program działał, należy wpisać ścieżkę do obrazu (jpg / png / itp).**<br>
> **Można w programie zmienić znaki, które mają być używane do generowania, jak i ilość pikseli, które mają być zamienione na pojedyńcze znaki.**

> 1. Program działa w Visual Studio 2022
> 2. Program działa tylko na systemach Windows (z powodu implementacji System.Drawing)
> 3. Program wymaga zainstalowania pakietu NuGet "System.Drawing.Common"
