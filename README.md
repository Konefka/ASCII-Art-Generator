# ASCII-Art-Generator
This program reads an image, calculates the average color of its pixels, and prints it as characters in the console.
> [!NOTE]
> To use the program, paste the path to your image file (jpg, png, etc.).<br>
> You can customize which characters are used and how many pixels get converted into each character.
