# ASCII-Art-Generator
Program przetwarza obraz który został podany piksel po pikselu i wyświetla wynik w konsoli
> [!NOTE]
> **Aby program działał, należy wpisać ścieżkę do obrazu.**<br>
> **Można w programie zmienić znaki, które mają być użyte, jak i ilość pikseli, które mają być reprezentowane przez znaki.**

## Instalacja

### Otwórz terminal

```bash
cd (tam, gdzie chcesz przechować projekt)
git clone https://github.com/Konefka/RPG-Console.git
```

### Otwórz Eksplorator plików

```
- Przejdź do folderu ASCII-Art-Generator
- Otwórz, po czym włącz ASCII-Art-Generator.sln
```

> [!WARNING]
> 1. Jeżeli automatycznie nie włącza Ci się Visual studio:
>    - Naliśnij na RPG_Console.sln prawym przyciskiem - Otwórz za pomocą - Microsoft Visual Studio 2022
> 
> 2. Jeżeli wyskakuje Ci błąd, to jest szansa, że nie masz zainstalowanego .NET SDK 9.0
>    - Wtedy idź do https://dotnet.microsoft.com/en-us/download, zainstaluj i spróbuj jeszcze raz
